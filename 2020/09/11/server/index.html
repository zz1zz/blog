    传输文件命令:(替换host为你的域名或ip)

    1.从服务器上下载文件

    scp 用户名@host:文件的绝对路径 本机绝对路径
    例子：scp root@8.8.8.8:/root/test.txt /Users/mac/Desktop (ip为8.8.8.8的服务器路径/root/test.txt 中的test.txt文件下载到本地桌面/Users/mac/Desktop)

    2.上传文件到服务器

    scp 本机绝对路径 用户@host:绝对路径

    3.上传目录(在scp后面加-r)

    scp -r 文件夹的绝对路径 用户@host:保存文件夹的绝对路径
    例:scp -r test root@8.8.8.8:/root/ 把当前目录下的test目录上传到 ip为8.8.8.8的服务器的/root/ 目录
